1. SCROLL RESTORATION PROBLEM
-----------------------------
Problem: Jab page change karte hai to scroll position same rehta hai
Solution: ScrollToTop component add kiya gaya hai App.jsx me

Code Fix:
```jsx
import { useLocation } from 'react-router-dom';
import { useEffect } from 'react';

function ScrollToTop() {
  const { pathname } = useLocation();
  
  useEffect(() => {
    window.scrollTo(0, 0);
  }, [pathname]);
  
  return null;
}
```

2. VERCEL DEPLOYMENT ROUTING ISSUE
----------------------------------
Problem: Direct URL access (like /about, /careers) pe 404 error
Solution: vercel.json file add karna hoga

Create: vercel.json in root directory
```json
{
  "rewrites": [
    {
      "source": "/(.*)",
      "destination": "/index.html"
    }
  ]
}
```

4. BROWSER CACHE ISSUE (LAPTOP)
-------------------------------
Problem: Phone me kaam kar rha, laptop me nahi
Solutions:
- Hard Refresh: Ctrl+F5
- Clear Cache: Ctrl+Shift+Delete
- Incognito Mode: Ctrl+Shift+N
- Different Browser try karo
- DNS Flush: ipconfig /flushdns

5. VITE BUILD CONFIGURATION
---------------------------
Problem: Build time routing issues
Solution: vite.config.js me history fallback

```js
export default defineConfig({
  plugins: [react()],
  build: {
    rollupOptions: {
      output: {
        manualChunks: undefined,
      },
    },
  },
  server: {
    historyApiFallback: true,
  },
})
```
